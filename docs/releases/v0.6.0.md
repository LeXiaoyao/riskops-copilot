# v0.6.0 Model Lab Strategy Evaluation MVP

## Included

- **M6-A Strategy Scenario Schema** — `configs/strategy_scenarios.yaml`，5 个策略场景，每场景含 scenario_id、strategy_type、target_metric、levers、assumptions、constraints、compliance_boundary
- **M6-B Offline Strategy Evaluator** — `riskops/engines/model_lab/strategy_evaluator.py`，基于 M3 summary 做离线规则化估算，输出 baseline / scenario / delta / assumptions / limitations
- **M6-C ROI & Cost-Benefit Calculator** — `riskops/engines/model_lab/roi_calculator.py`，基于 demo cost assumptions 计算成本、收益、ROI、payback period
- **M6-E CLI Integration** — 新增 `scenarios` / `strategy-eval` / `roi` / `model-lab` / `render-model-lab` 五个命令
- Synthetic data boundary 和 demo estimate boundary 在所有输出中明确声明
- Full pytest pass（105 passed）

## CLI Quick Start

```bash
python scripts/validate_strategy_scenarios.py
python scripts/run_strategy_eval.py
python scripts/run_roi_calculator.py
python scripts/riskops_cli.py scenarios
python scripts/riskops_cli.py strategy-eval
python scripts/riskops_cli.py roi
python scripts/riskops_cli.py model-lab
python scripts/riskops_cli.py render-model-lab
```

## Known Limitations

- Rule-based offline demo estimate，不是因果推断，不是真实模型预测
- Static demo cost assumptions，不代表真实财务结果
- 不接真实数据源，无实时数据
- 无模型训练（M6-D 未实现）
- 无交互式 drilldown
