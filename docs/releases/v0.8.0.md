# v0.8.0 — M8 AI Agent + 话术引擎 + 合规 LLM

发布日期：2026-05-28

## 本版本交付

### M8-A DeepSeek Function Calling 动态数据查询
- `riskops/tui/tools.py`：10 个 pandas 查询函数，直接读取 synthetic_data/ Parquet
- `riskops/tui/tool_schemas.py`：10 个 JSON Schema，供 DeepSeek FC 使用
- `riskops/tui/chat_client.py`：`stream_chat_with_tools`，多轮 FC，工具执行结果回传

### M8-B Textual TUI 对话终端
- `riskops/tui/app.py` 接入 Agent Orchestrator
- 工具调用事件橙色高亮（#f59e0b），Agent 路由蓝色加粗
- CLI 入口：`python scripts/riskops_cli.py tui`

### M8-C Agent Orchestrator + 4 专家 Agent
- `riskops/agents/orchestrator.py`：关键词路由（合规/话术/报告/风险）
- `risk_analyst.py`：10 工具 FC，动态查询底层数据
- `collection_strategy.py`：话术推荐链路，提取案件 ID
- `compliance_qa.py`：本地关键词质检，无需 API Key
- `report_writer.py`：Excel / PPT / MD / HTML 报告路由

### M8-D 话术推荐引擎
- `riskops/engines/script/case_context.py`：P0-P2 字段加载，P4 字段不进入 LLM
- `riskops/engines/script/frequency_checker.py`：频次检查（SMS 日≤2/周≤5，AI 外呼日≤3/周≤10）
- `riskops/engines/script/script_engine.py`：10 种话术模板 + 合规扫描 + LLM 润色
- `riskops/engines/script/mock_approval.py`：Mock 审批日志追加 jsonl
- CLI：`python scripts/riskops_cli.py script --case-id CASE00000001 --channel sms`

### M8-E QC LLM 11维合规评分
- `riskops/engines/qc/llm_scorer.py`：DeepSeek 11维评分 + `_fallback_score`
- `merge_with_keyword_scan`：关键词红线强制覆盖（合规红线≤20，high总分≤30）
- `compliance_scanner.scan_text_with_llm`：两路融合入口
- CLI：`python scripts/riskops_cli.py qc-scan --texts "话术" --use-llm`

### M8-F PPT 报告
- `riskops/engines/report/ppt_renderer.py`：生成 `m4_business_report.pptx`
- CLI：`python scripts/riskops_cli.py render-ppt`

## 边界说明
- LLM 输出仅供参考，不做自动策略决策，所有动作需人工确认
- P4 字段（姓名/手机/身份证/地址）不进入 LLM 上下文
- 所有功能基于合成数据，不代表真实业务能力

## 测试
- 新增测试：`tests/test_llm_scorer.py`（15个），`tests/test_orchestrator.py`，`tests/test_tools.py`，`tests/test_tui.py`，`tests/test_script_engine.py`，`tests/test_visualization.py`
- 全测试集：216 个测试，全部通过
