# v0.5.0 CLI Interaction MVP

## Included

- Unified CLI entrypoint: `python scripts/riskops_cli.py`
- `summary` command — project status and M3/M4 output overview
- `anomalies` command — high-priority anomaly list with severity, baseline, recent value, relative change
- `drivers` command — M1 D7 recovery rate decline top 5 drivers with business interpretation and recommended actions
- `outputs` command — dashboard, report, and M3 output file paths with existence status
- `render-dashboard` command — re-render `outputs/dashboard/dashboard.html`
- `render-report` command — re-render `outputs/reports/m4_business_report.md` and `.html`
- Synthetic data boundary explicitly displayed in `summary` output
- Business interpretation boundary in `drivers` output (attribution signals ≠ final root cause)
- Full pytest pass (67 passed)

## Not included

- Real customer data integration
- LLM-generated analysis
- Agent orchestration
- Collection action automation
- SMS / voice / WhatsApp integration
- Interactive TUI drilldown
- Heavy TUI framework

## Known limitations

- Text CLI only, no interactive drilldown
- No real-time data source
- Synthetic data only
- No LLM-generated narrative
- No real collection action
